# 校内成绩分析系统

本目录是一个只做校内成绩分析的本地静态系统。直接打开 `index.html` 即可使用，不需要登录、云端或部署。

## 使用入口

- 主页面：`index.html`
- 成绩模板：页面右上选择年级后，在“导入”页下载。
- 任课模板：页面“导入”页下载；支持“班级、学科、教师姓名、教师编号”窄表，也支持常见的宽表任课表。教师编号不是必填，但校内有同名教师时建议填写。
- 参考名单：页面“导入”页可选导入学籍/参考名单，用于核对名单内未出成绩、成绩表多出学生、缺考/转入转出等情况。
- 导出页可保存当前考试为历史基准，后续导入新考试后可做班级与学生进退步对比。

## 年级与学科口径

- 6、7、8 年级：沿用原 `school-system` 的 6-8 年级全科总口径，但总分只纳入当前年级满分配置里的学科；额外识别到的体育、科学等列只做单科诊断，不混入总分。
- 8 年级：物理满分 100 分，化学满分 100 分，模板已包含两科。
- 9 年级：沿用原 `school-system` 的五科总口径，只把语文、数学、英语、物理、化学计入总分；政治、历史、体育等保留单科诊断和教师分析，但不计入五科总。
- 优秀线：9 年级取年级前 15%，6-8 年级取年级前 20%。
- 达标线：取年级前 50%；系统另列“卷面及格率”，按对应满分的 60% 计算。
- 两率一分权重：6-8 年级为 60/70/70，9 年级为 50/80/50。
- 学科雷达图按“均分 / 满分”的得分率展示，避免 150 分科和 50 分科直接用原始分比较。
- 导入成绩会先保留原始分，再按当前年级满分参与统计；导入页“计算口径”可确认或修改原始满分。8 年级物理、化学默认原始满分 100；9 年级物理、化学默认原始满分 100，因此 9 年级物理按 100→90 折算，化学按 100→60 折算，不再依赖导入最高分猜测。
- 空白成绩默认按 0 分计入本科均分，以匹配当前月考口径；导入页可切换为“空白/缺考剔除”或“空白剔除，缺考按 0 分”，导出文件会记录该口径。
- 重复导入同一学生时，系统按学校、班级、考号优先合并；没有考号时按学校、班级、姓名合并，避免重复 sheet 或重复选择文件导致人数和均分放大。若重复记录同一学科分数不同，导出文件会生成“重复冲突”工作表。
- 总分缺少应计学科的学生不再把空白科目当 0 分参与总分均分、总分排名和班级总分评价；班级表会同时显示实际人数和总分有效人数。
- 同一学科同时出现“一卷、二卷、总、折合”等多列时，系统优先取最终成绩列：优先折合/折算，其次总分/总/成绩；如果某个学生最终列空白但分卷有分，会用分卷合计兜底。
- 当前年级只分析当前年级班级号的学生；同一学校文件里混入其他年级时会自动排除，无法识别年级的班级保留分析。
- 表头含“折合/折算”的成绩列视为已经折算到配置满分，不会再按原始满分二次折算；如需强制按原始满分折算，可在“计算口径”手动输入原始满分。
- 导入后会显示“成绩列映射预览”。系统默认优先取折合/折算列，其次取总分列，再用分卷合计兜底；如某次表头特殊，可在映射预览里手动指定采用列并立即重新计算。
- “导入前校验”会在正式分析前检查总分科目、缺科、重复冲突、超满分、参考名单和任课匹配；总分科目存在阻断问题时不会导出正式结果。
- “计算口径”里的配置满分和总分科目可临时调整，适合月考、期中、一模等不同考试方案；可保存并复用考试方案模板，“恢复默认口径”会回到当前年级默认方案。
- “评价权重”可在导入页调整并锁定。班级综合分、教师总榜都会把当前权重写入导出文件，便于追溯。

当前满分配置：

- 6-8 年级：语文 150、数学 150、英语 150、政治 100、历史 50、地理 50、生物 50；8 年级物理 100、化学 100。
- 9 年级：语文 150、数学 150、英语 150、物理 90、化学 60、政治 100、历史 60、体育 50。9 年级总分只计语数英物化。

## 教师最终排名逻辑

1. 先按同一学科比较，不把不同学科直接混排。
2. 每个教师学科项先计算“两率一分”质量分。
3. 用学生“其他科总分”估算本学科预期分，实际高于预期则给基础校正，低于预期则扣减。
4. 如果已保存历史考试，会按学生跨考试匹配后的本学科进步幅度做历史校正；没有历史基准时该项为 0。
5. 任课表可选填“任课权重/任课比例/课时比例”。合班、协同任课、半班授课时，教师最终汇总按有效学生权重计算，避免工作量被放大。
6. 样本量小不会放大成绩，工作量只做小幅修正，低分率单独扣分。
7. 教师最终总榜以学科质量分为主，学科内百分位和历史进步校正按可配置权重辅助，再按任教学生数、任课权重和置信系数加权汇总，避免同学科人数很少时仅靠名次拉开过大。
8. 如果导入文件包含多所学校，系统会要求在顶部选择当前分析学校，避免同名班级跨校混算。
9. 任课表允许表头前有标题行；也支持“班级、班主任、语文、数学、英语……”这种宽表，并会补齐 Excel 合并单元格里的任课教师。未匹配到当前学校班级或本学科成绩的任课记录只进入“未匹配”提醒，不进入教师排名。
10. 教师总榜默认只纳入当前总分科目；政治、历史、体育等仍保留教师学科明细。教师页可手动勾选进入总榜的学科，导出会记录该口径。
11. 任课表带“教师编号/教师工号/手机号”等列时，同名教师会按编号拆开统计；没有编号的教师会在诊断中提示，防止同名教师被误合并。
12. 任课表可选填“任课范围、学生考号、学生名单”。如果某科是走班、分层或小组授课，系统会只按对应学生计算该教师，不再默认整班覆盖。

班级评价在单校场景下使用“班级综合分”：相对两率一分、满分得分率达成、学科均衡度共同参与，并对低分率做小幅扣分，避免只看总分均分或只看校内相对名次。

学生页可直接导出“学生各科与总分排名”，包含每科折算后分数、每科校排、每科班排、总分校排和总分班排。

班级页提供“班级下钻”和“成绩分布”。班级下钻会列出薄弱学科、重点关注学生和样本风险；成绩分布会显示均分、标准差、四分位数和优秀/良好/及格/低分段。总览页还会显示总分优秀、良好、及格、低分四段人数图。

教师页提供“教师排名解释”，可拆开查看某位教师最终分的学科项分、权重、基础校正和风险提示。样本少、总分有效人数偏低、低分率高、分组任课等情况会在页面和导出中标记。

学生页可点击“报告”查看个人诊断，并导出个人报告。报告包含各科分数、校排、班排、距优秀线/达标线、得分率和优势/短板判断。

导出的综合工作簿包含“成绩折算”“成绩分布”“导入诊断”“导入前校验”“计算追溯”“学生个人报告”“考试对比”和“算法口径”工作表；如发现任课记录未匹配，会额外包含“任课未匹配”工作表；如发现重复导入分数冲突，会额外包含“重复冲突”工作表；如发现同名教师、未填教师编号或教师总榜排除学科，会额外包含“教师诊断”工作表。导出表格会自动设置筛选、列宽和百分比格式，方便直接查看。

导出页还提供“报告版”工作簿、可打印 HTML 报告和匿名版 Excel。匿名版会隐藏学生姓名、考号和教师姓名，只保留稳定匿名编号、班级、分数和排名。

本地保存使用 IndexedDB，容量比 localStorage 更适合大成绩表；浏览器不支持 IndexedDB 时会自动退回备用存储。导出页“清除本地保存”只清理浏览器留档，不会删除电脑上的源 Excel 文件。

## 验证

已通过：

```powershell
npm run smoke
```

烟测分为逻辑烟测、浏览器烟测和真实样例回归。逻辑烟测覆盖手动列映射、考试方案、走班任课、成绩分布、学生报告、历史考试对比和 IndexedDB 基础写入；浏览器烟测会打开本地页面、载入示例数据、切换 8/9 年级、检查教师排名、检查图表渲染，并做一次移动端宽度检查；回归烟测会读取桌面真实 `月考0427.xlsx` 和任课表，复核 9.6 语文均分 95.09、9 年级总分满分 600、宽表任课解析，以及参考名单、任课权重、历史校正、超满分阻断和计算追溯。

本次修复额外验证了：

- 9 年级物理、化学即使导入最高分低于配置满分，也仍按确认的 100 分原始满分折算。
- 宽表任课表和合并单元格任课表均可展开到班级、学科、教师。
- 重复学生会合并；6-8 年级额外学科不会进入总分。
- 空白成绩、缺考成绩可按三种口径切换；重复学生冲突会记录明细。
- 混入其他年级的数据会被当前年级筛选排除；折合/折算列不会被二次折算。
- 最终成绩列空白时，可用该学生分卷列合计兜底。
- 教师编号可拆分同名教师；教师总榜科目可从默认总分科目改为自定义科目，也可清空总榜科目只看学科明细。
- 综合导出包含导入诊断、教师诊断，并对百分比列、列宽和筛选做了基础格式化。
- 成绩列映射可从自动识别改为手动指定列；考试方案可调整满分和总分科目。
- 走班/分组任课可按学生考号或学生姓名限定教师覆盖学生。
- 学生个人报告、班级下钻、成绩分布、教师解释、报告版导出和考试历史对比均可生成。
- 真实文件 `月考0427.xlsx` 中 9.6 语文均分为 95.09；`初中部任课教师一览表2025.09.05.xlsx` 宽表可解析出 230 条任课映射，并生成当前 9 年级教师学科明细。
- 参考名单导入、导入前校验、考试方案模板、评价权重锁定、教师任课权重、历史基础校正、计算追溯、HTML 报告、匿名导出、隐私清理和真实样例回归测试均已纳入验证。
