# 校内成绩分析系统

本目录是一个只做校内成绩分析的本地静态系统。直接打开 `index.html` 即可使用，不需要登录、云端或部署。

## 使用入口

- 主页面：`index.html`
- 成绩模板：页面右上选择年级后，在“导入”页下载。
- 任课模板：页面“导入”页下载，必需列为“班级、学科、教师姓名”。

## 年级与学科口径

- 6、7、8 年级：沿用原 `school-system` 的 6-8 年级全科总口径，导入识别到的学科都参与总分。
- 8 年级：物理满分 100 分，化学满分 100 分，模板已包含两科。
- 9 年级：沿用原 `school-system` 的五科总口径，只把语文、数学、英语、物理、化学计入总分；政治、历史、体育等保留单科诊断和教师分析，但不计入五科总。
- 优秀线：9 年级取年级前 15%，6-8 年级取年级前 20%。
- 达标线：取年级前 50%；系统另列“卷面及格率”，按对应满分的 60% 计算。
- 两率一分权重：6-8 年级为 60/70/70，9 年级为 50/80/50。
- 学科雷达图按“均分 / 满分”的得分率展示，避免 150 分科和 50 分科直接用原始分比较。
- 导入成绩会先保留原始分，再按当前年级满分参与统计；若某科导入最高分明显超过配置满分，系统会推断原始卷面满分并自动折算、封顶。例如 9 年级物理 100 分制导入会按 100→90 折算，化学 100 分制导入会按 100→60 折算，五科总分不会超过 600。
- 总分缺少应计学科的学生不再把空白科目当 0 分参与总分均分、总分排名和班级总分评价；班级表会同时显示实际人数和总分有效人数。

当前满分配置：

- 6-8 年级：语文 150、数学 150、英语 150、政治 100、历史 50、地理 50、生物 50；8 年级物理 100、化学 100。
- 9 年级：语文 150、数学 150、英语 150、物理 90、化学 60、政治 100、历史 60、体育 50。9 年级总分只计语数英物化。

## 教师最终排名逻辑

1. 先按同一学科比较，不把不同学科直接混排。
2. 每个教师学科项先计算“两率一分”质量分。
3. 用学生“其他科总分”估算本学科预期分，实际高于预期则给基础校正，低于预期则扣减。
4. 样本量小不会放大成绩，工作量只做小幅修正，低分率单独扣分。
5. 教师最终总榜以学科质量分为主，学科内百分位只作小幅辅助，再按任教学生数和置信系数加权汇总，避免同学科人数很少时仅靠名次拉开过大。
6. 如果导入文件包含多所学校，系统会要求在顶部选择当前分析学校，避免同名班级跨校混算。
7. 任课表允许表头前有标题行；未匹配到当前学校班级或本学科成绩的任课记录只进入“未匹配”提醒，不进入教师排名。

班级评价在单校场景下使用“班级综合分”：相对两率一分、满分得分率达成、学科均衡度共同参与，并对低分率做小幅扣分，避免只看总分均分或只看校内相对名次。

学生页可直接导出“学生各科与总分排名”，包含每科折算后分数、每科校排、每科班排、总分校排和总分班排。

导出的综合工作簿包含“成绩折算”工作表；如发现任课记录未匹配，会额外包含“任课未匹配”工作表，方便回头修正班级或学科名称。

## 验证

已通过：

```powershell
npm run smoke
```

烟测会打开本地页面、载入示例数据、切换 8/9 年级、检查教师排名、检查图表渲染，并做一次移动端宽度检查。
